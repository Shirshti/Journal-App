# Habit Arena

Habit Arena is a habit tracking application built using Spring Boot, MySQL, HTML, CSS, and JavaScript.

## Features

* JWT Authentication
* User Registration & Login
* Create Habits
* Mark Habits as Completed
* Streak Tracking
* Leaderboard System

## Tech Stack

* Java
* Spring Boot
* Spring Security
* JWT
* MySQL
* HTML
* CSS
* JavaScript

## Future Improvements

* React Frontend
* Deployment
* Better UI/UX
* Analytics Dashboard
